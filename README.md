## 📖 프로젝트 개요

이 프로젝트는 안드로이드(태블릿) 환경에서 **Wittibox & Wittibus 앱 테스트를 자동화**하기 위한 전용 툴입니다.  
주요 목표는 메인 콘텐츠 실행 및 재생 여부를 효율적으로 검증하는 것입니다.

이 툴은 다음과 같은 기능을 제공합니다:
- 🎯 **UI 자동화** : Airtest 기반 화면 요소 탐지 및 터치/스크롤 동작 수행
- 🖼 **이미지/텍스트 인식** : OCR 기반 화면 검증
- 📊 **로그 및 리포트 생성** : 실행 중 Log 출력, 테스트 결과를 Excel 리포트로 자동 생성
- 🔄 **ADB 연동** : 실제 디바이스와 유/무선 연결을 통한 테스트 실행
